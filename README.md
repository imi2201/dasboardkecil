## Setup Environtment (in Windows)
```
  python -m venv myenv
  # Activate the virtual environment
  myenv/Scripts/activate
  # Install library(s)
  pip install numpy pandas scipy matplotlib seaborn jupyter streamlit babel
```
  ##Run Streamlit
    streamlit run dashboard.py
